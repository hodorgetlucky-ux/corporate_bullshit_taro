# 🔮 Корпоративное Таро · Corporate Tarot

Расклад Таро на корпоративные передряги. Чтобы в нервотрёпке и неопределённости
было за что зацепиться и на душе полегчало. Не коммерция, чисто по фану.

A three-card Tarot spread for workplace messes — for a bit of support and calm
when things get chaotic. Not a business, just for fun.

## Что умеет

- Поле для ввода ситуации **+ голосовой ввод** (Web Speech API).
- Кнопка «Раскинуть карты» → тянет три случайные карты из полной колоды (78 карт,
  прямые и перевёрнутые).
- ИИ-толкование расклада применительно к твоей ситуации — честно, по-простому,
  но всегда заканчивается на тёплой поддерживающей ноте.
- Два языка: **русский и английский** (переключатель в шапке).
- Тёмная тема в духе Slack с магической атрибутикой.

## Как устроено

- **Фронт:** Vite + React + TypeScript.
- **Бэк:** одна serverless edge-функция на Vercel — [`api/reading.ts`](api/reading.ts).
  Она и только она ходит в Anthropic API.
- **Колода:** [`src/deck.ts`](src/deck.ts) — карты тянутся на клиенте, а в API
  уходят только их названия и положение.

## 🔒 Безопасность ключей

API-ключ Anthropic **никогда не попадает в браузер**:

- Запрос к Claude делает только серверная функция `api/reading.ts`, читая ключ из
  `process.env.ANTHROPIC_API_KEY`.
- В коде фронтенда ключа нет, и он не передаётся клиенту.
- `.env`, `.env.local` и `.vercel` в [`.gitignore`](.gitignore) — секреты не уедут в git.
- Ключ хранится только в переменных окружения Vercel.

## Деплой на Vercel

1. В Vercel создай проект из этого репозитория (Framework preset: **Vite**).
2. В **Settings → Environment Variables** добавь:
   - `ANTHROPIC_API_KEY` = твой ключ Anthropic
3. Нажми **Deploy** (или Redeploy, если ключ добавил после первого билда).

## Локальная разработка

```bash
npm install
# создай .env.local с ключом:
#   ANTHROPIC_API_KEY=sk-ant-...
vercel dev      # чтобы работала и serverless-функция /api/reading
```

`npm run dev` поднимет только фронт без API — расклад будет рисоваться, но
толкование не придёт. Для полноценной локальной работы нужен `vercel dev`.
